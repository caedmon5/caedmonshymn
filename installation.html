Available Versions

§ I.1    There are several versions of the text available on this CD-ROM:

  • HTML Display Text (start.htm). This version can be viewed using an ordinary web browser and works on all operating systems.
  • Indexed Display Text (Windows 9x/NT/2000/XP only; requires installation to hard disk). This version is identical to the HTML Display Text but has been indexed for full-text searching using the Greenstone Digital Library program. A self-install program for this version is included on the CD-ROM. See installation instructions below.
  • SGML Master Files (See html/sgm/readme.txt). This version contains the source files used to produce the display text. It is not intended for direct display.

Startup

Windows

§ I.2    Windows users can start the edition simply by placing the CD-ROM in the CD drive of their computer. An auto-run feature will then present the available options for use and/or installation. If the auto-run feature fails, users may start the menu by selecting windowslaunch.exe from the root directory of the CD-ROM or by following the instructions for Other Operating Systems (below).

Menu Options (Windows only)

§ I.3    The opening screen of the Windows version presents users with the following options:

  • Run from CD-ROM (no search function)
  • Install Searchable Version to Hard Disk
  • Browse CD-ROM
  • Browse Fonts
  • System Requirements
  • Installation Instructions
  • Exit
Run from CD-ROM (no search function)

§ I.4    This option loads the HTML Display Text into the default browser. This is the same as choosing index.htm.

Install Searchable Version to Hard Disk (Windows 9x/NT/2000/XP only)

§ I.5    This option installs the Indexed Display Text to the user's hard disk. See Installation below for an overview of the installation options and procedure. Once this version is installed, users may access it from the Windows "Start" button (Start > All Programs > Caedmon's Hymn > Caedmon's Hymn).

Browse CD-ROM

§ I.6    This option allows users to browse the file system of the CD-ROM directly.

Browse Fonts

§ I.7    This option provides direct access to the fonts folder.

System Requirements

§ I.8    This option opens the "System Requirements" chapter in the default browser.

Installation Instructions

§ I.9    This option opens the "Installation Instructions" chapter (this file) in the default browser.

Exit

§ I.10    This option exits the Windows menu program.

Installation

Fonts

§ I.16    All electronic versions of this edition require the installation of the Junicode fonts to the system hard disk. These fonts are found in the fonts/ directory. While other fonts on your system may contain most of the necessary characters, the use of Junicode is recommended.

§ I.17    Installation of fonts varies from operating system to operating system. In most cases, it involves copying the font files to a fonts directory (you can find this by searching your disk for files ending in the extension .ttf). Some systems also have a utility to help in the installation process. In Windows, for example, this can be found in the "Control Panel." Users should consult their system manual for precise instructions on font installation. Troubleshooting instructions and suggestions can be found in the fonts readme file.

Indexed Display Text (Windows 9x/NT/2000/XP only)

§ I.18    The Indexed Display Text uses the Greenstone Digital Library program (Greenstone) to provide full-text searching of the edition on Windows 9x and later systems. The program is installed on the system hard disk using a Windows self-install program. After installation, the text of the edition can be accessed from the Windows Start menu (Start > All Programs > Caedmon's Hymn > Caedmon's Hymn).

Installation procedure

§ I.19    The self-install program can be accessed from the opening menu (select "Install Searchable Version to Hard Disk") or by double clicking Setup.exe in the CD-ROM's root directory. The following instructions refer to options offered by the Greenstone Installer program:

  1. The first screen asks the user to choose a directory for the installation. Accept the default installation directory (recommended) or enter the name of the directory to which you wish the indexed version of Cædmon’s Hymn to be installed. Select Next.
  2. If the directory supplied in step 1 does not exist, a pop-up window will appear asking if you want the installer to create it. Select Yes.
  3. The next screen asks about the installation of the "collection" (i.e. indexed text of Cædmon’s Hymn).
    1. The first option checks whether the user wishes to install the entire text of the edition to the hard drive. If the Install all collection files check box is checked, Greenstone will install the entire indexed text of the edition to the user's hard drive. This requires approximately 192 MB of free disk space and means that the user can consult the indexed text without the CD-ROM. If the Install all collection files check box is left blank, only the Greenstone program is installed. This requires approximately 30 KB of disk space. In this case, the user must have the CD-ROM available in order to consult the indexed text.
    2. As in step 1, above, the user may accept the default directory for the installation or supply a new path.
  4. Click on the Finish button to finish the installation.
  5. The installer will now ask you if you wish to install Netscape 4.05. Select No. This version of Netscape is distributed with Greenstone in order to ensure backwards compatibility with older systems. It will not work properly with Cædmon’s Hymn. See "System Requirements" for a list of acceptable browsers.
  6. The installation is now complete. To access the indexed version, use the Windows Start button.
