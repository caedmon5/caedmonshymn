HTM (/caedmon/html/htm/readme.txt)

This directory system contains HyperText Markup (HTML) files used to
display the contents of the edition in your browser. Files for the
most part are coded using the XHTML strict 1.0 and all files have been
validated.

To view these files you will need a Unicode aware browser. See
/system.htm for specific requirements.

The directory system also contains manuscript facsimile issues used in
the edition. These are found in the manuscript transcription
directories: /caedmon/html/htm/transcription/<manuscriptsigil>. The facsimiles for
Winchester, Cathedral I, for example, can be found in the directory 
/caedmon/html/htm/transcription/w/.
